Add users and specify their roles. Each must log in with their unique credentials. You can add users, specify their roles and control their permissions

Create users

  • Sign in to the web portal
  • Click on ‘retail’ and ‘user management’ and select ‘users’ 
  • Click ‘+’ to add a new user and provide the required information
  • Select desired role and assign to user

Create roles

  • Go to the web portal
  • Click on retail then user management and select roles
  • Click ‘+’ to add a new role and provide the required information
  • Select the desired permissions and save
